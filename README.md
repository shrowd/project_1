  - teapot, spoon i cup to pliki zawierające współrzędne punktów potrzebnych do narysowania powierzchni
- plik gra.py zawiera grę
  - space, spaceship, explosion i meteor to obrazy wykorzystywane w grze
- plik main.py jest to gui do łatwego uruchamiania zadań
